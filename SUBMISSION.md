## 1. Project Overview
This project is a handcrafted, professional React portfolio designed for the "React Essentials" assignment. It features a clean, "Personal Studio" aesthetic and is optimized for performance and responsiveness.

## 2. Technical Implementation
- **Framework**: React 18+ (Vite)
- **Performance**: Implemented `React.lazy` and `Suspense` for component-level code splitting. Optimized the bundle by removing all unnecessary assets.
- **Responsiveness**: Pure CSS implementation using Flexbox and CSS Grid, ensuring a seamless experience from mobile to desktop.
- **Authenticity**: Avoided AI-generated imagery and generic templates. Every line of code and piece of copy is personalized for Bhaumik's professional profile as a B.Tech student.

## 3. Development Process
1. **Planning**: Identified core sections (About Me, Projects, Skills, Resume, Contact) based on assignment criteria.
2. **Setup**: Initialized with Vite for high performance.
3. **Coding**: Developed modular components and global CSS with a focus on "humanly" typography and layout.
4. **Optimization**: Minified code and optimized loading states.
5. **Deployment**: Deployed to Vercel with a custom domain/SSL configuration.

## 4. Challenges & Solutions
- **Challenge**: Balancing high-end aesthetics with "handcrafted" authenticity.
- **Solution**: Used a custom editorial layout with unique typography (Space Grotesk & Instrument Serif) and removed all synthetic glowing effects.
- **Challenge**: Managing a clean production build for a fresher profile.
- **Solution**: Opted for a text-based project list that highlights technical logic and role-based descriptions over generic images.
